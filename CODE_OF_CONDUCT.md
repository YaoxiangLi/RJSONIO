# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

Project participants are expected to be respectful, constructive, and focused
on the work. Contributions, issue reports, and reviews should discuss code,
documentation, tests, reproducibility, and package behavior.

Unacceptable behavior includes harassment, personal attacks, discriminatory
language, or repeated disruption of technical discussion.

Report concerns to the package maintainer listed in `DESCRIPTION`.
